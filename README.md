# Тест-кейс 2: Создание клиента с валидными данными (Positive Test)

Предусловие: Форма отображается корректно (см. ТК-1)

Шаги:

1. Перейти в раздел "Add customer"
2. Заполнить все поля валидными данными 
3. Нажать кнопку Add Customer

Ожидаемый результат:

4. Клиент появился в таблице Customers

Фактический результат:

1. Все поля заполнены без ошибок валидации

3. Alert закрывается
4. Клиент появился в таблице Customers
