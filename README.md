# Тест-кейс 5: Успешная авторизация (Positive Tests)

Шаги:

1. Открыть страницу https://www.way2automation.com/angularjs-protractor/registeration/#/login
2. Заполнить следующие поля валидно:
   - Username (angular)
   - Password (password)
   - Username* (angular)
3. Нажать кнопку "Login"
4. Убедиться, что авторизация прошла успешно
5. Выйти из аккаунта

Ожидаемый результат:

1. Появляется страница с текстом "You're logged in!!"

Фактический результат:

----------

# Тест-кейс 6: Авторизация с валидным заполнением только двух полей (Negative Tests)

Шаги:

1. Открыть страницу https://www.way2automation.com/angularjs-protractor/registeration/#/login
2. Заполнить следующие поля валидно:
   - Username* (angular)
   - Password (password)
3. Поле "Username" заполнить рандомными данными (например, kokakola)
3. Нажать кнопку "Login"
4. Убедиться, что авторизация не произошла

Ожидаемый результат:

1. Появляется текст "Username or password is incorrect"
